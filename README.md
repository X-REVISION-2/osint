# UHC Osint
A chromium based osint platform, designed for linux.

## Dependencies
dig, whois, python3, chromium

## Python Deps
flask, hashlib, subprocess, os, sys, webbrowser, time, filetype, PIL

## Running
Just run the python file. That'll handle everything.

## Running with the binary
Make sure you have chromium installed, as it will not work without it. The binary is only 
available on linux.
